# Charcoal

Webapp to create and share posts (basically a less advanced [reddit](https://www.reddit.com/). )

![](images/frontpage.png)

# Features
 - Make new posts
 - Edit posts
 - Delete posts
 - Comment on posts
 - Edit comments
 - Delete comments
